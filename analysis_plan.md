# 분석 계획서: Clinical Motor Assessment Dashboard for Parkinson's Disease

## 한줄 요약
> 파킨슨 환자의 스마트워치 센서 데이터를 **정량적으로 시각화**해서, 의사가 더 정확하게 진단·평가할 수 있도록 돕는 **임상 의사결정 지원 대시보드**

---

## 1. 왜 이걸 만드는가? (문제 정의)

### 현재 문제
파킨슨 환자의 운동 기능 평가는 의사의 **관찰과 경험**에 크게 의존한다:
- 짧은 진료 시간에 떨림, 리듬, 좌우 차이를 한꺼번에 판단해야 한다
- "좀 떨리는 것 같다", "리듬이 불규칙한 것 같다" → **주관적 판단**
- 지난 방문과 비교하려면 의사의 기억이나 짧은 메모에 의존
- UPDRS 점수를 매길 때 정량적 근거가 부족

### 우리의 해결책
스마트워치 센서 데이터에서 **떨림, 리듬, 진폭, 좌우 비대칭** 등을 자동 추출하고,
건강한 사람 대비 얼마나 비정상인지, 지난번 방문 대비 어떻게 변했는지를
**숫자와 그래프로** 보여줌으로써 의사의 진단 정확도를 높인다.

### 핵심: 이것은 AI 진단기가 아니다
- AI가 "이 환자는 파킨슨이다"라고 판단하는 게 **아니다**
- 의사가 이미 알고 있는 것을 **더 정확한 숫자로 확인**하게 해주는 도구
- 최종 판단은 항상 **의사**가 한다

## 2. 어떤 데이터를 쓰는가?

### 데이터셋 1: PADS (스마트워치 센서 데이터)
- **출처**: PhysioNet (`parkinsons-disease-smartwatch/1.0.0`)
- **참여자**: 469명 (PD 환자 + 감별진단군 + 건강대조군)
- **센서**: 양손 Apple Watch (가속도계 + 자이로스코프, 100Hz)
- **운동 과제 11가지**: 안정 시 떨림, 팔 뻗기, 손가락 터치, 코 터치, 보행 등
- **추출할 특징**:
  - 떨림 강도 (tremor power)
  - 좌우 비대칭 (asymmetry index)
  - 리듬 불규칙성 (rhythm irregularity)
  - 움직임 크기 감소 (amplitude decrement)
  - 움직임 불안정성 (jerkiness/instability)

### 데이터셋 2: TULIP (영상 + 임상 평가)
- **출처**: Zenodo (record 14262223)
- **6대 카메라** 멀티뷰 영상 → 3D 움직임 궤적 복원 가능
- **활용**: 영상 기반 keypoint 궤적 → Motion Trail 시각화

## 3. 대시보드 전체 구성 (8개 탭)

```
┌──────────────────────────────────────────────────────────────────┐
│  Clinical Motor Assessment Dashboard for Parkinson's Disease     │
│  [환자 선택 ▼]  [과제 선택 ▼]  [PD Badge]                        │
├──────────────────────────────────────────────────────────────────┤
│ Tab1 │ Tab2 │ Tab3 │ Tab4 │ Tab5 │ Tab6 │ Tab7 │ Tab8           │
│ ┌────────────────────────────────────────────────────────┐      │
│ │                                                        │      │
│ │              메인 시각화 영역                             │      │
│ │                                                        │      │
│ └────────────────────────────────────────────────────────┘      │
└──────────────────────────────────────────────────────────────────┘
```

---

### Tab 1: Patient Overview (환자 요약)
**의사의 질문**: "이 환자 전체적으로 어떤 상태야?"

- **요약 카드**: Motor Score, Asymmetry, Tremor, Rhythm, Non-Motor Score, 가장 비정상인 과제
- **레이더 차트**: 5개 운동 특징 프로파일 한눈에
- **Top 비정상 과제**: 어떤 과제에서 가장 문제인지 바로 확인
- **해석 문구**: 자동 생성된 한줄 요약

### Tab 2: Task-Feature Heatmap (과제별 이상 패턴)
**의사의 질문**: "어떤 과제에서 어떤 종류의 문제가 있어?"

```
                 떨림  진폭감소  리듬불규칙  좌우비대칭  불안정
Finger tapping   🟥    🟧       🟥        🟧        🟨
Gait             🟨    🟧       🟧        🟥        🟥
Rest tremor      🟥    🟩       🟩        🟨        🟨
```
- 셀 클릭 → 상세 설명 표시
- 한 화면에서 6개 과제 × 5개 특징 = 30개 정보를 한번에 확인

### Tab 3: Left-Right Mirror Plot (좌우 비교)
**의사의 질문**: "양쪽 손/발 차이가 얼마나 커?"

- 좌우 양방향 바 차트 (파란색=왼쪽, 빨간색=오른쪽)
- Asymmetry Index 수치 + 정상군 대비 percentile
- **임상 의미**: 파킨슨은 보통 한쪽이 더 나쁨 → 비대칭이 진단의 핵심 단서

### Tab 4: Rhythm & Amplitude Panel (리듬 + 진폭)
**의사의 질문**: "반복 동작이 불규칙한가? 점점 작아지는가?"

- **시계열**: 좌/우 진폭 변화 + 검출된 피크 마커
- **탭 간격 차트**: 반복 동작의 간격이 얼마나 일정한지
- **진폭 감소 차트**: 시간이 지남에 따라 동작이 작아지는 정도
- 수치: CV(변동계수), Amplitude Decrement %

### Tab 5: Normative Comparison (정상군 대비 비교) ← NEW
**의사의 질문**: "이 환자가 정상에 비해 얼마나 비정상이야?"

- **분포 그래프**: 건강대조군의 분포(히스토그램/박스플롯) 위에 환자 위치를 빨간 점으로 표시
- 각 feature별로 "상위 몇 %"인지 표시
- **임상 의미**: "떨림이 좀 있다"가 아니라 "같은 나이대 건강인의 상위 92%에 해당하는 떨림"

### Tab 6: Patient History (경과 추적) ← NEW
**의사의 질문**: "지난번 방문 때보다 나빠졌어? 좋아졌어?"

- **방문별 추이 그래프**: 각 feature 점수가 방문 1→2→3 어떻게 변했는지
- 색상으로 악화(빨강) / 호전(초록) / 변화없음(회색) 표시
- **변화율 카드**: "Tremor +15%", "Rhythm -8%" 등
- **임상 의미**: 치료 효과 판단, 질병 진행 속도 모니터링

### Tab 7: UPDRS Score Assistant (점수 추정 보조) ← NEW
**의사의 질문**: "UPDRS 몇 점으로 매겨야 해?"

- 센서 데이터 기반으로 각 UPDRS 항목별 **추정 점수**를 제시
- 게이지 차트: 0~4점 스케일 위에 추정 위치 표시
- **각 항목별 근거**: 어떤 센서 수치가 이 점수를 뒷받침하는지 표시
- **주의**: "추정치"임을 명확히 표시 (최종 판단은 의사)

### Tab 8: Motion Trail (움직임 궤적)
**의사의 질문**: "실제로 손이 어떤 경로로 움직였어?"

- 손끝/손목의 2D 궤적 (색상 = 속도)
- PD 환자: 불규칙하고 떨리는 궤적
- 건강인: 매끄럽고 규칙적인 궤적

## 6. 핵심 메시지 (발표용)

이 대시보드는 파킨슨 환자를 **자동 진단하는 AI가 아니다**.
의사가 환자의 운동 기능을 평가할 때, **정량적인 근거**를 시각적으로 제공해서
더 **정확하고 일관된 판단**을 내릴 수 있도록 돕는 **임상 의사결정 지원 도구**이다.

- 센서 데이터 → 숫자로 변환 → 주관적 판단을 보완
- 정상군 대비 비교 → "얼마나 비정상인지" 객관적 기준 제공
- 경과 추적 → 치료 효과, 질병 진행 모니터링
- UPDRS 추정 → 점수 매기는 데 참고 자료 제공
